## 1.75.0 - 2023-12-28

### New Features

- command: new `targets` section in `firedbg.toml` to provide params to debugg targets
```toml
[[targets]]
# `quicksort/src/bin/quicksort.rs` (Binary)
name = "quicksort_100"
target.type = "binary"
target.name = "quicksort"
argv = ["100", "--seed", "1212"]
```
